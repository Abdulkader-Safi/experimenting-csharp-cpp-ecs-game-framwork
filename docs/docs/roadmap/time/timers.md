# Timers

Countdown and interval timers for cooldowns, spawning, and delays.

## Planned Scope

- Timer component (duration, elapsed, repeat flag)
- Countdown timer (fires once when expired)
- Interval timer (fires repeatedly at fixed intervals)
- Timer system that ticks all active timers each frame
