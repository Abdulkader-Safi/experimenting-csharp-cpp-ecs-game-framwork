| SPACE             | 32        |     | APOSTROPHE (')    | 39        |
| COMMA (,)         | 44        |     | MINUS (-)         | 45        |
| PERIOD (.)        | 46        |     | SLASH (/)         | 47        |
| **0-9**           | **48-57** |     | SEMICOLON (;)     | 59        |

| RIGHT_BRACKET (]) | 93        |     | GRAVE_ACCENT (\`) | 96        |
